API é um acrônimo para "Application Programming Interface". Apesar do nome "interface", não há nada de visual ou focado no usuário, é uma forma simples de comunicação entre aplicações. Não tem telas, suas respostas geralmente são em JSON (JavaScript Object Notation). Independe do front-end. São baseadas em requisições para rotas (endpoints) e respostas, geralmente assíncronas.

REST significa "Representational State Transfer" ou "Transferência Representacional de Estado". É um estilo de arquitetura que define como APIs devem ser criadas, seguindo todos os padrões, a API é considerada RESTful. Alguns dos pontos são: respostas uniformes, sem ligação de estado, cache de respostas, dentre outros.

SPA significa "Single Page Application", ou "Aplicação de Página Única". Ainda que o usuário clique em um link, estará apenas promovendo uma alteração de DOM (Document Object Model), não há recarregamento. Elas têm front-end separado do back-end, ou seja, uma API para o back-end e um framework para o front-end, os que dominam o mercado atualmente são React, Vue e Angular, mas são todos "passageiro". Aplicações do tipo estão dominando o mercado, em comparação ao antigo modelo "Monolith", em que um software possuía front-end, back-end e banco de dados, tudo em uma estrutura só.

Verbos HTTP são de suma importância para se compreender as APIs e fazem parte do REST. Há os métodos "GET", "POST", "PUT", "DELETE" e "PATCH". Cada um representa uma ação na API: o GET recupera os dados da rota, o POST insere dados na rota, o PUT faz a atualização total dos dados na rota, o DELETE apaga os dados na rota e o PATCH realiza uma atualização parcial ou total nos dados da rota, sendo uma opção menos arriscada que o PUT. É muito importante que as funcionalidades usem os métodos corretos. Envia-se o método através da requisição.

Uma API é desenvolvida de forma muito semelhante a quando criamos projetos com Handlebars, por exemplo. Temos algumas alterações leves, como resposta apenas por JSON. Mas isso acaba até simplificando as coisas, separando as responsabilidades.
